Instrucciones de uso para el makefile:

-Para compilar todas las clases:

$ make

-Para ejecutar el programa:

$ make run

-Para ejecutar todos los drivers:

$ make runTests

-Para ejecutar todos los JUnits:

$ make runTestsJUnit

-Regla de cada driver:

$ make testDriverJugador

$ make testDriverKakuro

$ make testDriverCasilla

$ make testDriverCapaDominioJugar

$ make testDriverBibliotecaKakuro

$ make testDriverCapaDominioKakuro

-Para remover los archivos compilados:

$ make clean
